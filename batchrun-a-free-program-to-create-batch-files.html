A Free Program To Create Batch Files

Are you looking for an easy to use way to create batch files without using complicated scripting languages? There are some programs out there that can handle it but most of them are pretty hard to use. One we have found is called Batchrun, and it is actually quiet easy to use. It allows you to create batch files in .brs that can do pretty much anything.

With Batchrun you can do things like run programs, end and kill processes, create directories, and delete directories. You can also change the priority of tasks, choose a run mode, and set a delay for comments. It has a simple interface that anyone can figure out how to use.

Batchrun makes your life easier by allowing you to make everything quicker. It does a lot of different things. While some batch files can only run programs, the files created in this app can do almost anything. It works on 32-bit and 64-bit versions of Windows XP, Windows Vista, Windows 7 and Windows 8.

Features:

  • Create batch files in .brs format.
  • Files can run programs, end and kill processes, create directories, and delete directories.
  • Supports all major Windows versions.

Find Batchrun @ outertech.com/en/batch-file
